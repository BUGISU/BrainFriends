# 언어평가 데이터 정리본

- `aq_norm_sd.csv`: 정상군 AQ 평균/표준편차 기준표(연령/교육년수별).
- `spontaneous_speech_anchors.csv`: 스스로말하기(내용전달/유창성) 점수 앵커.
- `result_entry_template.csv`: 케이스 결과 입력 템플릿.

## 계산 메모
- 유창성 점수(0~10) = (대화과제 점수 + 해변가과제 점수) / 2
- 스스로말하기 총점(0~20) = 내용전달 점수 + 유창성 점수

※ 참고: PDF OCR/텍스트 추출 특성상 일부 문구는 원문 의미를 유지하는 범위에서 정리되었습니다.
